## 七、潜在机会与差异化

### 核心要求

从竞品情报和蓝海发现两个维度，挖掘产品的差异化机会。本章节需要回答"我们可以在哪里赢"的问题。

### 分析框架

#### 1. 竞品情报分析
- **竞品提及统计**：用户评论中提及了哪些竞品品牌？各被提及多少次？
- **正面对比**：用户认为我们在哪些方面优于竞品
- **负面对比**：用户认为竞品在哪些方面优于我们
- **竞品替代意愿**：有竞品使用经验的用户，对我们产品的整体评价如何

#### 2. 定位矩阵分析
构建"满意度 x 提及频率"四象限矩阵：

```
高满意度 + 高提及 = 核心优势（继续强化）
高满意度 + 低提及 = 隐藏价值（需要更多曝光）
低满意度 + 高提及 = 核心短板（紧急修复）
低满意度 + 低提及 = 次要问题（持续监控）
```

#### 3. 蓝海细分场景发现
- 是否存在小众但满意度极高的人群？他们的共同特征是什么？
- 是否存在未被充分满足的使用场景？
- 是否有用户提出了创新的使用方式或需求？

#### 4. 差异化机会评估
- 基于竞品对比，找出我们的独特价值主张（UVP）
- 评估每个差异化机会的市场规模和竞争壁垒
- 提出差异化定位建议

### 必须包含的内容

- 竞品对比必须有具体数据支撑（提及次数、评价倾向）
- 必须引用评论原话，包含正反两方面
- 蓝海发现需说明判断依据和样本量
- 定位矩阵需以表格或图表描述形式呈现

### 格式参考

```
### 竞品对比：[竞品品牌]
**提及次数**：X 次
**我们领先的方面**：[列表]
**竞品领先的方面**：[列表]
**原话佐证**：
> "对比原话1"
> "对比原话2"

### 蓝海机会：[机会名称]
**目标人群**：[画像描述]
**样本支撑**：X 条评论提及类似需求
**市场潜力评估**：[分析]
**执行建议**：[建议]
```
