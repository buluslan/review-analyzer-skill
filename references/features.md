# 核心功能详细说明

## 22维度智能标签系统

工具提供22个维度的AI智能标签，全面覆盖评论信息：

### 标签维度

```
人群维度 (4): 性别、年龄段、职业、购买角色
场景维度 (1): 使用场景
功能维度 (2): 满意度、具体功能
质量维度 (3): 材质、做工、耐用性
服务维度 (5): 发货速度、包装质量、客服响应、退换货、保修
体验维度 (4): 舒适度、易用性、外观设计、价格感知
市场维度 (2): 竞品对比、复购意愿
情感维度 (1): 总体评价
```

完整的标签定义和说明，请查看：[tag_system.yaml](tag_system.yaml)

## 三模AI洞察系统

工具提供三种不同的AI执行环境，以便在不同的需求与成本下自由切换：

### Claude CLI本地模式
使用您的Claude Code中的内置模型进行全程任务执行。

### Gemini增强模式（推荐）
调用Gemini API，使用【Gemini 3.1 flash】生成洞察报告，使用【Gemini 3.1 pro】生成可视化看板（需要API Key，产生费用）

### Claude CLI+Gemini混动模式
文字报告使用Claude Code内置模型，可视化看板使用【Gemini 3.1 pro】生成可视化看板（需要API Key，产生费用）

详细说明：[dual_mode_system.md](dual_mode_system.md)

## 四位一体VOC系统

### 用户画像识别
工具会自动识别3-4类典型用户画像,例如:
- **实用主义者**: 关注性价比、耐用性
- **品质追求者**: 重视材质、做工、设计
- **科技发烧友**: 关注功能、性能、创新
- **时尚生活家**: 看重外观、体验、社交价值

### 3D头像系统
提供6套3D头像,增强用户画像的视觉表现力:
- 商务精英风
- 休闲生活风
- 技术极客风
- 温馨家庭风
- 银发族专属
- Z世代潮流

每类用户画像配备6条精选评论示例,构建立体化VOC系统。

详细说明：[voc_system.md](voc_system.md)

## 黑金奢华可视化看板

生成的HTML报告采用董事会高管汇报级别的黑金奢华设计。

### 视觉特性
- **色彩方案**: 黑色背景 + 金色点缀
- **发光效果**: 鎏金渐变创作者署名
- **交互图表**: 6个Chart.js交互式图表
- **响应式设计**: 支持各种屏幕尺寸

### 图表内容
1. **用户画像分布** - 饼图展示各类用户占比
2. **情感分析趋势** - 折线图展示评分分布
3. **维度标签云** - 词云展示高频标签
4. **功能满意度矩阵** - 热力图展示功能vs满意度
5. **使用场景分布** - 柱状图展示场景频次
6. **市场对比雷达** - 雷达图展示竞品对比

详细说明：[dashboard_features.md](dashboard_features.md)
