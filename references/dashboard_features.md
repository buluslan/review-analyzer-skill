# 黑金奢华可视化看板

## 设计理念

HTML 可视化看板采用**董事会高管汇报级别**的设计风格，以黑金配色为主，打造专业、高端的数据展示体验。

### 设计语言

- **色彩方案**: 黑色背景 (#000000) + 金色点缀 (#D4AF37)
- **视觉风格**: 奢华、专业、高端
- **交互体验**: 流畅、直观、易用
- **响应式设计**: 支持桌面、平板、手机

## 交互式图表

工具使用 **Chart.js** 库创建 6 个交互式图表，所有图表都支持：

- **鼠标悬停**: 显示详细数据提示
- **图例点击**: 显示/隐藏数据系列
- **缩放功能**: 支持数据区域缩放
- **导出功能**: 可导出为 PNG/SVG

### 1. 用户画像分布饼图

展示各类用户画像的占比分布。

**数据来源**: VOC 系统中的用户画像分类

**交互功能**:
- 点击扇区查看该类用户的详细信息
- 悬停显示具体数量和百分比
- 支持扇区高亮

### 2. 情感分析趋势折线图

展示评分分布的趋势变化。

**数据来源**: 所有评论的评分分布

**交互功能**:
- 悬停显示评分区间的评论数
- 支持时间序列切换（如果有时间数据）
- 可选择显示不同用户画像的趋势

### 3. 维度标签云词图

以词云形式展示高频标签。

**数据来源**: 22维度标签的词频统计

**交互功能**:
- 词的大小表示频率高低
- 悬停显示具体标签和出现次数
- 点击筛选包含该标签的评论

### 4. 功能满意度矩阵热力图

以热力图展示功能维度 vs 满意度的关系。

**数据来源**: 功能_具体功能 × 功能_满意度的交叉统计

**交互功能**:
- 颜色深浅表示满意度高低
- 悬停显示具体数量
- 支持按功能或满意度筛选

### 5. 使用场景分布柱状图

展示各使用场景的评论数量分布。

**数据来源**: 场景_使用场景的统计

**交互功能**:
- 悬停显示具体数量和占比
- 支持按场景筛选评论
- 可选择显示不同的用户群体

### 6. 市场对比雷达图

多维度竞品对比分析。

**数据来源**: 市场_竞品对比 + 质量/体验/服务维度

**交互功能**:
- 多个产品/品牌的雷达图叠加
- 悬停显示各维度的具体评分
- 支持添加/删除对比对象

│  │           │  │  ┌─────┐ ┌─────┐ ┌─────┐              │  │
│  │ 总评论数  │  │  │ 饼图 │ │折线图│ │词云图│              │  │

### KPI 卡片

显示关键指标：

| 指标 | 说明 |
|------|------|
| Total Reviews | 总评论数 |
| Tagged Reviews | 已打标评论数 |
| Average Rating | 平均评分 |
| Persona Count | 用户画像数量 |

### 图表网格

6 个图表以 3x2 网格排列：
- 第一行: 饼图、折线图、词云图
- 第二行: 热力图、柱状图、雷达图

## 高级功能

### 数据筛选

看板支持按以下维度筛选：
- **用户画像**: 选择特定用户群体
- **时间范围**: 选择特定时间段
- **评分范围**: 选择特定评分区间
- **标签维度**: 选择特定标签值

### 数据导出

支持导出格式：
- **PNG**: 高清图片
- **SVG**: 矢量图
- **PDF**: 完整报告
- **Excel**: 原始数据

### 数据刷新

- 手动刷新: 点击刷新按钮
- 自动刷新: 可设置自动刷新间隔
- 数据对比: 支持历史数据对比

## 使用场景

### 高管汇报
- **场景**: 季度汇报、年度汇报、董事会汇报
- **优势**: 专业、高端、一目了然
- **效果**: 提升汇报质量，增强说服力

### 客户展示
- **场景**: 向客户展示分析结果
- **优势**: 视觉冲击力强，易于理解
- **效果**: 提升客户满意度，增强信任

### 产品评审会
- **场景**: 产品迭代评审、用户体验评审
- **优势**: 数据驱动，可视化呈现
- **效果**: 提升评审效率，支持决策

### 市场分析报告
- **场景**: 市场调研报告、竞品分析报告
- **优势**: 专业、全面、深入
- **效果**: 支持市场决策，洞察市场机会
