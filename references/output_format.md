# 输出文件详细说明

## 三位一体输出

工具会生成三种格式的分析结果，满足不同使用场景。

## 1. CSV标签数据

**文件名格式**: `评论采集及打标数据_{ASIN}.csv`

**内容结构**:
- 原始评论数据（保留所有原始列）
- 22维度 AI 标签列（新增）

**22维度标签列**:
```
人群_性别, 人群_年龄段, 人群_职业, 人群_购买角色
场景_使用场景
功能_满意度, 功能_具体功能
质量_材质, 质量_做工, 质量_耐用性
服务_发货速度, 服务_包装质量, 服务_客服响应, 服务_退换货, 服务_保修
体验_舒适度, 体验_易用性, 体验_外观设计, 体验_价格感知
市场_竞品对比, 市场_复购意愿
情感_总体评价
```

**用途**:
- 数据分析：导入 Excel、Tableau、Power BI 等工具进行二次分析
- 统计报告：制作自定义报表和图表
- 筛选过滤：按特定标签维度筛选评论
- 趋势分析：对比不同时间段或产品

**示例**:
```csv
内容,评分,人群_性别,人群_年龄段,场景_使用场景,功能_满意度
这个产品质量很好,值得推荐!,5,女性,30-40岁,家用,超出预期
物流很快,包装也很仔细,5,不明,不明,不明,符合预期
```

## 2. Markdown洞察报告

**文件名格式**: `分析洞察报告_{ASIN}.md`

**内容结构**:
```markdown
# 评论深度洞察报告

## 执行摘要
- 分析概述
- 关键发现

## 战略机会点分析
- 市场机会识别
- 产品差异化建议

## 用户痛点总结
- 高频痛点分类
- 紧急改进项

## 产品优化建议
- 功能改进建议
- 服务提升建议

## 用户画像洞察
- 典型用户类型
- 各画像需求特点

## 市场定位洞察
- 竞争优势
- 差异化定位
```

**用途**:
- 汇报分享：直接用于团队汇报和管理层汇报
- 团队协作：作为产品改进和市场营销的讨论基础
- 决策支持：为产品迭代和战略调整提供数据支持
- 客户沟通：向客户展示分析结果和建议

## 3. HTML可视化看板

**文件名格式**: `可视化洞察报告_{ASIN}.html`

**功能特性**:
- 黑金奢华设计风格，适合高管汇报
- 6个交互式 Chart.js 图表
- 响应式设计，支持各种屏幕尺寸
- 纯 HTML 文件，可直接在浏览器中打开

**6个图表**:
1. **用户画像分布饼图** - 展示各类用户占比
2. **情感分析趋势折线图** - 展示评分分布趋势
3. **维度标签云词图** - 高频标签可视化
4. **功能满意度热力图** - 功能 vs 满意度矩阵
5. **使用场景柱状图** - 场景频次统计
6. **市场对比雷达图** - 多维度竞品对比

**用途**:
- 高管汇报：董事会级别的专业展示
- 客户展示：向客户呈现分析结果
- 产品评审会：可视化支持产品决策
- 市场分析报告：专业的市场分析呈现

**交互功能**:
- 鼠标悬停查看详细数据
- 图例点击显示/隐藏数据系列
- 响应式布局适配不同设备
